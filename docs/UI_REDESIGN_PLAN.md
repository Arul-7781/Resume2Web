## ğŸ¯ Implementation Strategy

### Phase 1: Template Structure
Create separate HTML templates for each design:
- `template_split_screen.html`
- `template_single_scroll.html`
- `template_sidebar.html`
- `template_bold_hero.html`
- `template_card_grid.html`
- `template_elegant.html`
- `template_dark_tech.html`

### Phase 2: Components Library
Build reusable components for each design:
- Navigation variants (top bar, sidebar, hamburger)
- Hero sections (split, full-width, video background)
- Project cards (grid, list, slider)
- Skills display (tags, progress bars, icon grid)
- Contact forms (inline, modal, full-page)

### Phase 3: Template Selection
Update portfolio model to include design preference:
```python
design_style: str = Field(
    default="split-screen",
    description="Portfolio design: split-screen, single-scroll, sidebar, bold-hero, card-grid, elegant, dark-tech"
)
```

### Phase 4: Smart Template Matching
AI suggests design based on profession:
- **Designer/Creative:** split-screen, bold-hero
- **Developer:** sidebar, dark-tech
- **Freelancer:** single-scroll, card-grid
- **Consultant/Coach:** elegant, bold-hero
